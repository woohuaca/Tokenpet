# 阿在 Token Pet 使用条款说明

## 当前分发性质

阿在 Token Pet 当前以：

- 公开仓库
- 公开下载包
- 本地 Codex 安装

的方式分发。

这不代表已经进入官方公共 Codex 插件目录。

## 当前适用范围

当前正式首发范围是：

- Codex
- macOS
- 本地插件入口 + 桌面 companion

## 当前不承诺的事项

当前版本不承诺：

- 所有宿主与所有系统都可用
- 已具备公开插件商店级别的安装体验
- 已具备自动备份和恢复能力

## 用户使用前应知悉

使用当前版本意味着你理解并接受：

- 它会读取本机 Codex Token 相关事件来驱动宠物喂养
- 它会在本地保存运行状态和日志
- 重置会清空当前宠物的成长和关系

## 后续完善

如果后续要提交到官方目录或官网页面，建议把这份说明整理成稳定外链，再写入：

- `interface.termsOfServiceURL`
