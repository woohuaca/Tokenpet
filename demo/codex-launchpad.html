阿在 Token Pet · Codex 启动页

安装后，先从这里把它叫出来。

这只桌宠吃的是你真实花掉的 Token。当前版本最简单的使用方式，是先启动它，再让它自然吃到一点 Token， 然后轻拍一次完成认主。

1. 点启动桌宠
2. 等它浮到桌面上
3. 吃到一点 Token 后轻拍认主
想先记住你的手感
启动桌宠

优先点这里。会拉起桌宠本体，并由它自行唤起 Token bridge。

 关闭桌宠

保留当前存档与成长状态，下次继续养这一只。

 重置当前这只

会清掉当前成长、关系和粮仓，请只在确认要重新生成时使用。

如果当前环境不允许直接执行

有些 Codex 界面或系统配置可能不会直接放行 `.command` 文件。这时请直接打开插件根目录里的这些入口：

插件根目录：加载中…
启动入口：加载中…
关闭入口：加载中…
重置入口：加载中…
如果刚才点了没反应：
1. 直接在 Finder 打开上面的 `start.command`。
2. 或回到 Codex 插件根目录，双击 `start.command`。
3. 看到桌宠后，再回到 Codex 继续工作即可。

第一次启动后做什么

  • 先让它吃到一点真实 Token
  • 再轻拍一次，完成认主
  • 之后它会根据饮食结构和关系慢慢长成自己的样子
