# 阿在 Token Pet 隐私说明

## 当前版本收集和读取什么

阿在 Token Pet 当前以本地运行为主。

在 `Codex + macOS` 首发范围内，它会读取或生成这些本地数据：

- `~/.codex/sessions` 中与 Token 使用相关的本机会话事件
- 插件目录 `runtime/` 下的宠物状态、配置、事件缓存与本地日志

## 当前版本不会默认做什么

当前版本默认不会：

- 自动把宠物运行态上传到外部服务
- 自动把会话内容同步到远端数据库
- 自动把本地 Token 事件转发给第三方平台

## 本地保存的数据类型

主要包括：

- 宠物状态
- 粮仓与成长记录
- 本地事件消费游标
- bridge 运行日志
- 桌宠显示位置等本地配置

## 用户可以如何清除数据

你可以通过以下方式清理：

- 关闭桌宠：`stop.command`
- 重置当前宠物状态：`reset.command`
- 手动删除插件目录下的 `runtime/`
