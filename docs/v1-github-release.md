# 阿在 Token Pet `v1.0.0` GitHub Release 文案

## Release Title

`v1.0.0` 阿在 Token Pet Codex 首发正式版

## Release Body

```md
阿在 Token Pet `v1.0.0` 发布了。

这是阿在 Token Pet 以 `Codex + macOS companion` 为首发范围的正式版发布。

## 这次发布是什么

- `Codex-first`
- `macOS companion`
- 本地可安装、可发现、可启动的正式首发版

这里的“正式发布”有边界：

- 已经按正式版口径进入 Codex 发布链
- 当前正式范围仍限定在 `Codex + macOS`
- 这不等于已经进入官方公共插件商店

## 你可以下载什么

### 1. `azai-token-pet-v1.0.0-public-share.zip`

适合：

- 公开下载分享
- GitHub 仓库访问者直接下载
- 想先拿完整插件包的人

### 2. `azai-token-pet-v1.0.0-codex-macos.zip`

适合：

- Codex 首发正式版分发
- 以 `Codex + macOS` 为目标环境的安装

## 当前已经成立的能力

- 真实 Token 会先转成粮食，再驱动宠物状态变化
- 宠物会自动吃粮，并逐步积累互动能源
- 首次诞生、认主、拍准、拍偏、错过已有明确反馈差异
- 单实例启动链与 bridge 自愈链已经收口

## 当前范围

- 宿主：Codex
- 系统：macOS
- 形态：插件入口 + 桌面 companion
- 启动入口：`start.command`

## 当前不包含什么

这次正式发布不代表：

- 已经完成多宿主正式接入
- 已经具备公开插件商店分发能力
- 已经做完安装器、备份恢复和完整图鉴宇宙

## 安装与说明

- 仓库首页：`README.md`
- 公开分发说明：`PUBLIC_DISTRIBUTION.md`
- 最短安装说明：`docs/v1-release-install-quickstart.md`
- 隐私说明：`PRIVACY.md`
- 条款说明：`TERMS.md`

## 校验

可在 `docs/v1-release-checksums.md` 查看本次发布包的 SHA-256。
```

## 建议上传的 Release Assets

4. `PRIVACY.md` 与 `TERMS.md` 是否都能公开访问
