# 阿在 Token Pet `v1.0.0` GitHub Release 5 分钟清单

## 目标

5 分钟内把当前仓库变成“别人可下载”的公开 Release。

## 步骤

### 1. 打开新建 Release 页面

打开：

- `https://github.com/woohuaca/Tokenpet/releases/new`

### 2. 选择标签

选择：

- `v1.0.0`

如果页面要求填写标题，使用：

- `v1.0.0 阿在 Token Pet Codex 首发正式版`

### 3. 粘贴正文

直接复制：

- `docs/v1-github-release.md`

里的 `Release Body`

### 4. 上传两个附件

上传：

- `builds/azai-token-pet-v1.0.0-public-share.zip`
- `builds/azai-token-pet-v1.0.0-codex-macos.zip`

### 5. 检查校验值

如果需要一起贴 SHA-256，复制：

- `docs/v1-release-checksums.md`

### 6. 发布

确认后点击：

- `Publish release`

## 发布后立刻检查

1. Release 页面是否能看到两个 zip
2. 仓库首页 README 是否能解释下载和安装
3. `PRIVACY.md` 与 `TERMS.md` 是否可打开
4. Issues 页面是否出现 `Bug Report` 与 `Feature Request`

## 如果只想让别人最快下载

优先提醒他们下载：

- `azai-token-pet-v1.0.0-public-share.zip`
