# 阿在 Token Pet `v1.0.0` 最短安装说明

## 先下哪个包

如果你只是想最快跑起来，优先下载：

- `azai-token-pet-v1.0.0-public-share.zip`

如果你明确是为了 `Codex + macOS` 首发正式版环境，也可以下载：

- `azai-token-pet-v1.0.0-codex-macos.zip`

## 三步跑起来

1. 下载并解压 zip
2. 进入插件目录
3. 运行 `start.command`

## 系统要求

- macOS
- `xcrun`
- `python3`

## 启动后你会看到什么

- 桌宠会在桌面上出现
- 它会把真实 Token 转成粮食库存
- 它会自动吃粮并逐步积累互动能源
- 第一次建议先让它吃到一点 Token，再轻拍一次完成认主

## 常用操作

- 启动：`start.command`
- 关闭：`stop.command`
- 重置：`reset.command`

## 当前边界

- 当前正式范围限定在 `Codex + macOS`
- 当前还不是官方公共 Codex 插件目录分发
- 当前仍以本地脚本启动为主
